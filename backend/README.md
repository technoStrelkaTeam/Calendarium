# Тестовые логины и пароли пользователей в базе:
## Администратор:
логин -  SUPERADMIN
пароль - string

может получать всех пользователей, а также получить/удалить/обновить пользователя с таким-то {user_id}. Могут все что обычные пользователи

## Пользователи:
логин - boba777
пароль - qwerty

или

логин - masha70
пароль - s

могут получить/обновить/удалить информацию о себе.
